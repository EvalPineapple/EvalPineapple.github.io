Eval Pineapple

Eval pineapple replaces the boring eval rectangle on Chess.com and Lichess with faces of grandmaster Hikaru Nakamura. As white's advantage increases, Hikaru smiles more; as black's advantage increases, Hikaru frowns more.

Custom Images

If you want to use your own set of images, put them in a json file like this example, then click choose file in the top right. When you start making moves again they'll use your new set of images, based on the eval numbers you provide. Your json needs at least "mate", "lowest", and "mated" images, and can have number images (eg 5, 3, 2.2, 0, -1, -1.1, whatever you want). Each numbered image is used from its number to the number higher than it, so there image #3 would be used from eval 3 to eval 5 and image #5 would be used from eval 5 to eval mate.
